928 * of "bset".
929 *
930 * We first compute a unimodular transformation that puts the unbounded

932 * that maps all equalities to equalities (in HNF) on the first dimensions.
933 * Let x be the original dimensions and y the transformed, with y_1 bounded
934 * and y_2 unbounded.
935 *
936 * [ y_1 ] [ y_1 ] [ Q_1 ]
937 * x = U [ y_2 ] [ y_2 ] = [ Q_2 ] x

939 * Let's call the input basic set S. We compute S' = preimage(S, U)
940 * and drop the final dimensions including any constraints involving them.

945 * resulting in

950 * [ Q_1 ]
951 * [ F 0 ] [ Q_2 ] x >= g
